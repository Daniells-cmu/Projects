
Project Description:
	The game is a side scrolling game that transitions into a fighting stage. The game is only single player. There are 3 different bosses to beat depending on the difficulty of the level. Score is based on obstacles avoided, monsters killed, the number of lives left at the end and the difficulty level.

How To Run the Project:
	Choose the difficulty that you would like to play in on the main screen and then press the start button or the tutorial to learn the controls.

Libraries needed:
	Pygame and PIL

Short commands:
	You can press 6 while in the battleArena mode to make it to the top10 and move on to the stage clear screen
